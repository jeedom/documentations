# Desarrollo central

## Arborescence

El código está distribuido en diferentes directorios en la raíz de Jeedom *(`/var/www/html` por defecto)* :

- **Terceros** : Carpeta que contiene las bibliotecas externas utilizadas por Jeedom *(jQuery, CodeMirror, etc)*.
- **respaldo** : Carpeta de copia de seguridad de Jeedom.
- **centro** : Carpeta que contiene las funciones internas del núcleo:
	- **Ajax** : Archivos PHP que sirven de interfaz entre clases JS y clases PHP.
	- **API** : Archivos PHP de la API.
	- **clase** : Archivos de clase PHP *(eqLogic, cmd, jeeObject, history, cron, etc.)*.
	- **com** : Archivos de clase de comunicación PHP *(http, shell)*.
	- **configuración** : Archivos PHP de configuración principal y *default.config.ini* incluyendo la configuración predeterminada. Archivo de versión para la versión Core.
	- **CSS** : Iconos disponibles con el Core y su CSS.
	- **i18n** : Archivos JSON que contienen las cadenas traducidas.
	- **imagen** : Imágenes (logotipos, fondos, etc.) del Núcleo.
	- **js** : Archivos de clase JS, llamados en particular desde páginas de Jeedom.
	- **PHP** : Archivos PHP necesarios para el núcleo (excluyendo las clases)).
	- **repositorio** : Archivos PHP específicos para cada mercado, Samba, etc.
	- **plantilla** : Archivos HTML *(Panel de control y móvil)* para mostrar eqLogics (Mosaico), comandos (Widgets) y escenarios.
	- **temas** : Archivos CSS para los tres temas principales (Oscuro, Claro, Clásico), para el panel de control y la versión móvil.
- **datos** : Carpeta que contiene datos de usuario (informes, vistas, personalización avanzada CSS/JS, diseño 3D, etc.)).
- **de oficina** : Una carpeta que contiene todas las páginas que se muestran (la interfaz) en el escritorio y sus funciones.
