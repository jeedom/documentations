## Interface do usuário

### Design e ergonomia

A interface usa :
- **Gradientes de cor** Verde para o cabeçalho (consistência BACnet))
- **Painéis com gradiente** bege para áreas de configuração
- **Botões modernos** com efeitos de passar o mouse
- **Indicadores visuais** (distintivos, ícones) para o status da configuração

### Navigation

- **Botão de rolagem flutuante** : Permite chegar rapidamente ao final da página
  - Posição baixa por padrão
  - Ele sobe automaticamente quando atinge o nível mais baixo
  - ícone de seta para baixo

### Responsive

A interface se adapta a diferentes tamanhos de tela, mantendo todas as suas funcionalidades.

## Atalhos e dicas

### Atalhos de teclado

Atualmente, não existem atalhos de teclado implementados para esta tela.

### Astuces

1. **Duplicar um objeto** : Copie os valores manualmente para uma nova linha
2. **Importação rápida** : Utilize um arquivo JSON para importar vários objetos simultaneamente
3. **Teste rápido** : Use "Sincronizar agora" após cada alteração para verificar o comportamento
4. **Modo manual** : Útil para testar valores fixos antes de vincular um comando Jeedom
