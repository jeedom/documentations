Es ist wichtig, die eingereichten Änderungen klar zu erläutern, damit der/die Entwickler des ursprünglichen Projekts Ihre Anfrage verstehen und bestätigen können.

- Klicken Sie auf die drei kleinen Punkte
- Wir kopieren die zuvor eingegebenen Informationen
- Ebenso kopieren wir (daher die Verwendung von … in Schritt 1, um ein erneutes Schreiben zu vermeiden))
- Klicken Sie auf „Pull Request erstellen“

![tuto.github11](images/tuto.github11.png)

**Es ist vorbei.** Sie müssen warten, bis Ihr Pull Request validiert wurde.

NB: Nur Benutzer mit Push-Berechtigungen für Jeedom können den Pull Request bestätigen.

Um sicherzustellen, dass Ihre Änderung in der Liste erscheint, können Sie auf Pull Requests klicken

![tuto.github12](images/tuto.github12.png)

Wir erhalten die Liste der PRs, die auf die Validierung warten. Wir können unsere deutlich sehen
